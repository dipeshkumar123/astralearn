🚀 HOW TO RUN TESTS
═══════════════════════════════════════════════════════════════════════════════

Quick Commands:

1. Run all backend tests:
   cd server
   npm test

2. Run specific test:
   cd server
   npm test -- auth-security.test.js

3. Run integration tests:
   cd server
   npm run test:integration

4. Run all tests from root:
   node run-tests.js
   (or: bash run-tests.sh on Mac/Linux)

5. Run with coverage:
   cd server
   npm test -- --coverage
